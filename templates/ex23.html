Animation: Predicting Future Value of $10,000

Use JavaScript to implement the following features :

  • Create an animation that shows the value of $10,000 at any interest rate for each year in the future for a total of 10 years. The user will enter a number for the "Interest Rate" and clicking the "Start" button will initiate the animation.
  • The "Year" and "Value" should update every second. For example, after the first second, the "Year" should change to 2024 and the "Value" should change to $10,500.00 assuming an interest rate of 5%.
  • Every second, the "Year" should increase by 1 and the "Value" should increase based on the interest rate entered by the user. For example, after 10 seconds, the "Year" should be 2033 and the "Value" should be $16288.94 assuming an interest rate of 5%.
  • After reaching the year 2033, the "Year" should be reset to 2023 and the "Value" should be reset to $10000.00.
  • You must use the setTimeout function for the animation and always display the values with two decimal places using the toFixed method.
